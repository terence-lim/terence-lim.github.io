finds.database.sql.as_dtypes(df: DataFrame, columns: Dict, drop_duplicates: List[str] = [], sort_values: List[str] = [], keep: str = 'first', replace: Dict[str, Tuple[Any, Any]] = {}) → DataFrame[source]ÔÉÅ

Convert DataFrame dtypes to the given sqlalchemy Column types

Parameters:

  • df ‚Äì Input DataFrame to apply new data types from target columns

  • columns ‚Äì Target sqlalchemy column types as dict of {column: type}

  • sort_values ‚Äì List of column names to sort by

  • drop_duplicates ‚Äì list of fields if all duplicated to drop rows

  • keep ‚Äì ‚Äòfirst‚Äô or ‚Äòlast‚Äô row to keep if drop duplicates

  • replace ‚Äì dict of {column label: tuple(old, replacement) values}

Returns:

DataFrame with columns and rows transformed

Notes:

  • Columns of DataFrame are dropped if not specified in columns input

  • If input is None, then return empty DataFrame with given column types

  • Blank values in boolean and int fields are set to False/0.

  • Invalid/blank values in double field are coerced to NaN.

  • Invalid values in int field are coerced to 0
