LLM Tutorial

Learning Outcomes

After finishing this assignment, you should be able to say: 

  • I have some basic knowledge about LLM prompting methods such as in-context learning.

  • I know how to apply LLMs to health-related issues.

Rationale

The goal of this assignment is to get familiar with generative AI technologies (e.g., ChatGPT) and know how to apply them to understand healthcare issues.

Instructions

In this tutorial, you will demonstrate how you apply generative AI technologies using synthetic data or publicly available data on healthcare related issues. Some areas you might explore include (but are not limited to):

  1. Summarize medical information and evaluate the summarization;

  2. Convert tabular data into text using different prompt engineering methods (e.g., in-context learning, tree of thoughts, chain of thoughts, reasonings of GPT) for disease prediction and analysis;

  3. Utilize different LLMs in healthcare (such as PMC-LlaMA)

  4. Medical notes simplification for different patient groups. (Example)

This paper from Nature has several application areas using LLMs which contain related references and hopefully can inspire your idea for your LLM tutorial.

Once you have an idea of what you'd like to focus on, create prompts and use data (see note below) to see how you can apply LLMs to it. For full credit, you will need to try several in-context learning and few shot learning methods and reasoning methods. See the rubric below for more information. 

Note: Medical data can not be used or shared directly via openAI GPT API or the web interface. There are synthetic patient data that can be used with OpenAI GPT.

Here are some publicly available synthesized data that you can use with openAI to analyze:

  1. https://synthea.mitre.org/downloads 7 MB of data with 100 patients

  2. https://mitre.box.com/shared/static/9iglv8kbs1pfi7z8phjl9sbpjk08spze.zip 54MB of 10k COVID related data.

  3. https://github.com/bepnye/EBM-NLP (this dataset is a text corpus that can be used if we need only text as input)

  4. https://data.world/siyeh/synthetic-medical-data
