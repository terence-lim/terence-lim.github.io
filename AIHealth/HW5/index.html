MIMIC ML/DL

Create a tutorial about applying ML/DL using MIMIC data.

Learning Outcomes

After finishing this assignment, you should be able to say: 

  • I can apply ML/DL to EHR data.

  • I can apply ML/DL to identify basic problems in healthcare.

Rationale

The goal of this assignment is to get familiar with the machine learning and deep learning technologies and know how to apply them to analyze EHR data and derive insights. In this tutorial, you will demonstrate how you extract features and how to fit your features to the existing machine learning models that can provide predictions for risk management in healthcare. Or you can apply deep learning methods on multimodality data, temporal data, or other kinds of data to provide evidence-based understandings of diseases and delivered healthcare. 

Instructions

For this project, you will create a project that uses Machine Learning or Deep Learning methods to execute a task on existing data. At a minimum, you should be able to recreate the tutorial from Lecture Series 11

For full credit, you will need to use different ML/DL methods from our examples for different downstream tasks (e.g., not readmission or length of stay) and evaluate your results. See the rubric below for more information. 

Once you have completed your project, create a slide deck that goes through your process step-by-step and that includes at least one slide with your thorough evaluation of the results. 

Submission

  • A python file

  • Slides that explain the steps you used to create your project and an evaluation of your results
